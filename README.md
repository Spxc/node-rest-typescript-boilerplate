# Node 20 typescript boilerplate
Simple boilerplate for using `typescript` with `node20`

## Features
- Env checks with `zod` on boot
- Nodemon for `pnpm dev`
- Target `es2017`
- Folder target paths ie: `@config`
- Github actions for testing
